## 00-overview.md

```markdown
# 评论分析报告：[主题名]

**数据来源**: [平台名]，[N] 篇笔记，[M] 条评论
**分析日期**: YYYY-MM-DD
**数据时间范围**: YYYY-MM-DD ~ YYYY-MM-DD

## 核心发现

> 用 2-3 句话概括最重要的发现。这不是摘要，而是你的判断。

## 数据概览

| 指标 | 数值 |
|------|------|
| 分析笔记数 | |
| 总评论数 | |
| 一级评论 | |
| 二级回复 | |
| 评论最高点赞数 | |
| 信号分布 | 高频提问 X% / 场景追问 X% / 对立观点 X% / 决策问题 X% / 情绪表达 X% / 故事叙述 X% |

## 关键信号摘要

1. [最重要的信号，一句话 + 证据密度]
2. [第二重要的信号]
3. [第三重要的信号]

## 推荐优先级

| 优先级 | 选题方向 | 理由 |
|--------|----------|------|
| P0 | | 频率最高 + 情绪最强 |
| P1 | | 有争论空间 |
| P2 | | 细分场景覆盖 |
```
