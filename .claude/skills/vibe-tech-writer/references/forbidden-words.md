## 替换策略

| 违禁词 | 替换方式 |
|--------|---------|
| 然而 | 直接转折，用"但""可""不过"；或句号断开，下一句直接说反面 |
| 此外/另外 | 删掉，直接接下一句 |
| 综上所述/总之 | 删掉，用一个有力量的金句代替总结 |
| 值得注意的是 | 删掉，直接说值得注意的事 |
| 基于此/因此 | 用"所以""这才有了""这下""于是" |
| 第一/第二/第三 | 用关键词小标题代替编号，或用场景推进 |
| 显而易见/毫无疑问 | 删掉，直接陈述事实，让读者自己判断 |
| 尽管如此 | 用"话是这么说""道理是这个道理" |
| 换句话说 | 用"说白了""意思是" |
| 一方面...另一方面 | 删掉结构，直接讲两面，读者自然能感知对比 |
