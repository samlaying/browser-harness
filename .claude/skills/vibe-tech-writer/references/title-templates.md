# 4种爆款标题结构

## 结构A：趋势判断型

**公式**：[熟悉事物]，[正在成为/正在变成] + [宏大定位]

**特征**：
- 逗号制造停顿感
- "正在成为"暗示趋势未定、读者正见证
- 给一个不起眼的东西赋予战略高度

**示例**：
- "灵动岛，正在成为 AI 时代的新门面"
- "手帐，正在成为 vibe coding 时代的第一根扶手"
- "Mac 刘海，正在成为 AI Agent 的第一块屏幕"

---

## 结构B：功能爆炸+情绪型

**公式**：[产品名] + [动词] + 「夸张词」功能，[目标人群] + [情绪词]

**特征**：
- 「」括号强调核心卖点
- 锁定最大公约数人群
- 情绪词预设读者反应

**示例**：
- "Claude Code 发了个「王炸」功能，打工人狂喜"
- "Cursor 上了个「逆天」功能，程序员坐不住了"
- "Suno 推了个「作弊级」功能，音乐人破防了"

---

## 结构C：恐惧/好奇型

**公式**：[时间紧迫词]，[主语动态]，[你的XX正在成为XX]

**特征**：
- "刚刚"制造时效紧迫感
- "你的XX"直指读者本人
- "正在成为XX"暗含不安感或未知感

**示例**：
- "刚刚，Codex 大更新，你在电脑的操作正在成为 AI 经验包"
- "刚刚，GPT-5.5 上线，你写过的每一封邮件都在变成训练数据"
- "刚刚，苹果发布新系统，你手机上的灵动岛正在变成 AI 的脸"

---

## 结构D：数字反转型

**公式**：[具体数字]，[主语] + [反常识结论]

**特征**：
- 数字前置是最强注意力钩子
- 两个数字层层递进
- 反常识结论颠覆行业叙事

**示例**：
- "营收涨了 30 倍，ARR 3 亿美金，这家 AI 公司已经在赚钱了"
- "用户破了 1000 万，估值 0，这款 AI 工具拒绝融资"
- "日活 500 万，团队只有 3 个人，这个 vibe coding 项目吓到投资人"

---

## 标题检查清单

生成标题后逐项检查：
- [ ] 不超过30字？
- [ ] 命中至少一种情绪（惊讶/好奇/恐惧/惊喜）？
- [ ] 不包含"深度解析""全面解读""一文看懂"？
- [ ] 不包含违禁词？
- [ ] 读完会产生"我必须点进去"的冲动？
