# 终审自检清单

## 去AI化检查（一票否决）

- [ ] 全文扫描146个违禁词，零命中？
- [ ] 没有"第一/第二/第三"或"第一条/第二条/第三条"？
- [ ] 没有"随着XX的发展""在当今时代"开头？
- [ ] 没有"总的来说""简而言之"收尾？

## 标题检查

- [ ] 不超过30字？
- [ ] 命中至少一种情绪（惊讶/好奇/恐惧/惊喜/共情）？
- [ ] 不含"深度解析""全面解读""一文看懂"？
- [ ] 读完有"必须点进去"的冲动？

## 开头检查

- [ ] 3句话以内完成"画面→钩子→悬念"？
- [ ] 有至少2种五感还原？
- [ ] 有时间或空间锚点？
- [ ] 读完产生"然后呢？"的冲动？

## 结构检查

- [ ] 5模块完整（钩子→背景→核心→对比→升维）？
- [ ] 核心层小节标题是判断句，不是模糊描述？
- [ ] 小节之间不用"首先/其次/最后"推进？
- [ ] 核心层占全文40%左右？

## 情绪线检查

- [ ] 有4-6个情绪转折点？
- [ ] 开头是好奇/惊讶/恐惧/共情之一？
- [ ] 结尾是深思/恍然大悟/力量感之一？
- [ ] 相邻情绪点不重复？
- [ ] 三条建议/清单部分没有情绪平坦化？

## 五感与活人感检查

- [ ] 全文至少有5个五感还原点？
- [ ] 视觉和时间感在开头出现？
- [ ] 每个核心层小节有至少1个五感点？
- [ ] 全文用"你"而不是"用户""读者"？
- [ ] 至少有3处口语化表达？

## 金句检查

- [ ] 至少有3句值得截图的金句？
- [ ] 覆盖至少3种类型（判断/比喻/反转/升维/留白）？
- [ ] 结尾有1句有力量的收束金句？
- [ ] 每句不超过30字？

## 人设检查

- [ ] 全文人设一致（不中途变换叙事姿态）？
- [ ] 人设是"朋友聊天"或"行业观察者"，不是"老师讲课"？
- [ ] 有具体的人设语言标记（至少2处）？

## 节奏检查

- [ ] 有长句和短句的交替？
- [ ] 信息密集段后有场景描写减速？
- [ ] 用短句开场（第一句不超15字）？
- [ ] 不出现连续3段以上的纯论述（中间必须穿插场景或案例）？
